# Task 03 - Automated Login Test

This project automates login functionality testing of a web application using Selenium and Python.

## Tools Used
- Python
- Selenium
- PyTest
- Chrome WebDriver

## Test Cases Covered

1. Valid Login Test
2. Invalid Login Test
3. Empty Fields Validation

## Test Website
https://the-internet.herokuapp.com/login

## How to Run

Install dependencies

pip install -r requirements.txt

Run tests

pytest test_login.py
